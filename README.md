## 🎨 Overview

**AirDraw AI** is a state-of-the-art web application that completely redefines how you interact with a canvas. Instead of using a mouse or a touchscreen, you can literally draw in the air using hand gestures!

Powered by Google's **MediaPipe Hands**, it tracks 21 distinct 3D landmarks on your hand in real-time, providing incredibly precise and smooth drawing capabilities natively in your browser.

## 🚀 Features

- **Pinch to Draw:** The drawing mechanism is highly intuitive. Just pinch your index finger and thumb together to start drawing on the screen.
- **Hover to Move:** Release the pinch and move your open hand to reposition your cursor without leaving a mark.
- **Vibrant Color Palette:** Choose from a set of beautiful, neon-inspired colors.
- **One-Click Clear:** Quickly clear your masterpiece and start fresh using the minimalist UI controls.
- **Glassmorphism Design:** A premium, modern UI featuring frosted glass effects, dynamic gradients, and smooth animations.
- **No Installation Required:** Runs entirely in the browser using pre-trained ML models loaded via CDN.

## 🛠️ Tech Stack

This project was built with a focus on performance, leveraging vanilla web technologies to keep the bundle size minimal and execution blazing fast.

- **Frontend Build Tool:** Vite (Vanilla Template)
- **Computer Vision:** MediaPipe Hands API
- **Rendering:** HTML5 Canvas API
- **Styling:** Vanilla CSS3 (Custom Variables, Flexbox, Animations)

## 💻 Getting Started

Want to run AirDraw AI locally? It takes less than a minute.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sreeharivponline/AirDraw.git
   cd AirDraw
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5173/` and grant the browser permission to access your webcam.
